# 🖌 프로젝트 정적 파일

> *번들러를 통해 포함되지 않을 파일들이 여기에 담깁니다.*

번들러를 통해 번들링되면 이름이 임의로 바뀌며 내용도 축약됩니다. 또한 번들러를 통해서만 접근 할 수 있게 되므로, `favicon.ico` 나 `manifest.json` 같이 기존 값의 변화 없이 바로 전달되어야 하는 파일들은 `assets` 폴더가 아닌 `static` 폴더를 통해서 제공되어야 합니다.
